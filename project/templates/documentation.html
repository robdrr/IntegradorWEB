Guía del Analizador Hepático

1. Carga de Archivos

El sistema acepta archivos CSV con los siguientes formatos:

  • Datos de expresión diferencial Sano vs NAFLD
  • Datos de expresión diferencial Sano vs NASH
  • Datos de expresión diferencial NAFLD vs NASH

2. Análisis de Datos

El análisis incluye:

  • Conteo de genes diferencialmente expresados
  • Visualización de gráficos de volcán
  • Determinación automática del estado hepático

3. Visualización 3D

El sistema proporciona una visualización 3D del hígado basada en el estado determinado:

  • Sano: Representación de un hígado normal
  • NAFLD: Visualización de un hígado con esteatosis (acumulación de grasa)
  • NASH: Modelo de un hígado con esteatohepatitis (inflamación y daño celular)
